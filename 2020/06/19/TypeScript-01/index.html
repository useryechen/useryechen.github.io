静态语言与动态语言的对比

静态类型语言 动态类型语言
对类型极其严格 对类型非常宽松
立即发现错误 BUG隐藏时间久
运行效率高，性能好 运行效率低，性能差
自文档化 可读性差

类型

ES6数据类型

  • Boolean,
  • Number,
  • String,
  • Array,
  • Function
  • Object
  • null
  • undedfined
  • Symbol

TypeScript新增数据类型

  • void
    • 表示没有任何返回值的类型
  • any
    • 不指定变量类型，那么它的类型就是any类型，可以任意赋值
  • never
    • 永远不会有返回值的类型
  • 元组
    • 可以通过push方法添加成员
    • 通过push添加进去的成员无法访问
  • 枚举
  • 高级类型
类型注解
  • 作用：相当于强类型语言中得类型声明
  • 语法：(变量|函数): type
特殊的类型
  • undefined
  • null

在typescript中，undefined和null是任何类型的子类，不可以讲这两个类型赋值给其他类型，但是可以讲其他类型赋值给这两个类型。需要在编辑器中配置tsconfig.json,strictNullChecks: false在配合联合类型使用
